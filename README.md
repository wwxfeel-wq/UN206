# 时墨之境
一个写信比发消息更慢、却更真的地方 先和大家打个招呼 👋 我是一名动态设计师，日常工作就是和时间轴、关键帧、动效曲线打交道。职业习惯让我对「时间」和「情感」这两个东西特别敏感——动画的本质就是用时间去制造情绪，而我的工作就是让每一帧都「有感觉」。这个项目其实源于我一直思考的一个问题： "我们借助科技消除了等待，消除了相思之苦，可是，会不会有一种可能，情感本身，跟着也薄了呢？"

Demo 简介 是什么： 时墨之境（Time Ink Realm）是一款以「写信」为核心的情感陪伴应用。它不追求即时回复的快感，而是让用户在一个有温度的 AI 陪伴者「时墨」的引导下，慢慢写下心里说不出口的话。当情绪达到峰值时，还会收到一封来自陌生人的实体安慰信件——整个机制由 AI 实时评估。
面向谁：

🎯 核心用户：18-35 岁年轻人，心里有话说不出口，想找个安全的地方倾诉 🌊 延伸用户：异地恋情侣、想给父母写信的人、需要情感出口的都市打工人 ✨ 特色覆盖：情绪低谷期需要被「看见」的人、对实体信件有情怀的文艺青年 主要功能：

💬 对话 — 不是 AI 助手，是一个有脾气的人

动态情绪：根据对话内容实时切换情绪状态（喜悦/生气/温柔/担忧/调皮） 记忆系统：记住用户上次说过的话、没说完的话 态度鲜明：会直接指出用户在逃避，不是一味安慰 图片分享：支持 AI 智能搜图，聊天中自然发送猫猫、风景、表情包 点击放大：图片可全屏查看，支持 Lightbox 弹层 ✉️ 写信 — 慢下来，把心里话写下来

对话式引导：不是表单填写，而是像朋友聊天一样慢慢展开 写信流程：收件人 → 城市 → 称呼 → 信件主题 → 正文 → 寄出 已寄出信件管理：查看寄出记录和送达状态 收信箱：接收来自陌生人的信件 💙 情绪关怀系统 — 当你撑不住的时候，有人写了一封信给你

双类型情绪追踪：正向峰值触发祝贺信，负向峰值触发安慰信 情绪标记：AI 每轮回复末尾带 [mood:type:score]，前端解析并记录趋势 触发条件：连续 2 轮 ≥75 分 + 上升趋势 年度配额：正向 3 封/年，负向 6 封/年 月度峰值刷新：每月重置情绪追踪 24 小时冷却：触发后冷却期内不重复触发 后台统计：设置面板展示年度使用情况、进度条、上次触发日期 🔄 情绪触发流程： 用户聊天 → AI 回复 + 情绪标记 → 连续 2 轮 ≥75 分 → 配额检查 → 生成安慰信 → 弹出通知

微信消息秒回，视频通话随时连，我们已经没有「等一封信」的经历了。科技消除了等待，但等待本身就是情感的一部分。等信的那几天，你会反复想对方会怎么回应，会在信到的那天小心翼翼地拆开——这种「慢」，恰恰是情感最浓的时候。

"我们借助科技消除了等待，消除了相思之苦，可是，会不会有一种可能，情感本身，跟着也薄了呢？" — 这就是时墨之境存在的原因

想解决的问题：

用户痛点 现有方案的不足 我们的解法 心里有话说不出口 社交软件太公开，心理咨询太正式 AI 陪伴者引导式对话，像朋友一样陪你慢慢说 即时通讯让情感变薄 消息秒回，没有期待和等待的空间 写信寄出后要「走」几天，故意保留等待的仪式感 情绪低落时没人发现 朋友圈发了也没人注意，或怕打扰别人 AI 实时评估情绪，达到峰值时自动触发陌生人的安慰信 AI 对话太「工具感」 永远温和、永远积极，像客服一样模板化 有脾气、有记忆、有态度的真实人格，会生气也会心疼 聊天中想分享图片 手动搜索复制链接，体验割裂 AI 自然发图，写 [img:search:关键词] 自动搜图显示 为什么做这个方向：

📊 市场空白：目前没有一款产品把「情感陪伴 + 写信 + 情绪关怀」打通 💚 社会价值：都市年轻人普遍孤独，需要一个安全、私密的情感出口 🤖 AI 深度赋能：情绪评估、信件生成、智能搜图，AI 不是工具而是「人」 🎨 设计驱动：动态设计师的视角，让产品本身就是一种情感体验 🚀 可落地性强：用 TRAE 三天从 0 到 1 做出可交互的完整 Demo

💡 体验建议： 对话：和「时墨」聊天，试试说「今天好累」或「发张猫猫图」 写信：点击菜单 → 写信，体验对话式引导写信流程 收信箱：菜单 → 收到的信，查看来自陌生人的信 情绪关怀：连续说几句情绪低落的话（如「好累」「撑不住了」），观察是否触发安慰信通知 设置面板：菜单 → 设置，查看情绪关怀系统的年度统计面板
